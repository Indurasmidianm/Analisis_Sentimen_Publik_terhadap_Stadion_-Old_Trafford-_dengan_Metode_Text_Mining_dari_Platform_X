## 📌 Background
Old Trafford dikenal sebagai “Theatre of Dreams” karena menjadi markas bersejarah dari Manchester United F.C.. Stadion ini tidak hanya menjadi tempat berlangsungnya pertandingan sepak bola, tetapi juga simbol sejarah, budaya, dan identitas klub. Popularitasnya menjadikan Old Trafford pusat perhatian publik serta memunculkan berbagai opini dan diskusi di media sosial, khususnya pada platform X.

---

## ❓ Problem Statement
Bagaimana distribusi sentimen positif, negatif, dan netral dari tweet yang membahas Stadion Old Trafford?

---

## 👥 Stakeholder
Hasil analisis ini dapat dimanfaatkan oleh beberapa pihak, antara lain:

- Manajemen klub  
- Pengelola stadion  
- Pemerintah daerah  

Analisis sentimen dapat membantu dalam pengambilan keputusan, evaluasi kebijakan, serta peningkatan kualitas pengalaman bagi para penggemar sepak bola.

---

## 🛠️ Approach
Penelitian ini menggunakan pendekatan *Natural Language Processing* (NLP) untuk melakukan analisis sentimen terhadap tweet yang membahas “Old Trafford”. Tahapan analisis meliputi:

- Data collection dari platform X  
- Text preprocessing  
- Tokenization  
- Sentiment classification  
- Visualisasi distribusi sentimen  

Setiap tweet dikategorikan ke dalam sentimen positif, negatif, atau netral berdasarkan isi teks yang dianalisis.

---

## 📂 Dataset
Dataset yang digunakan terdiri dari beberapa atribut berikut:

| Column | Description |
|---|---|
| conversation_id_str | ID percakapan tweet |
| created_at | Waktu tweet dibuat |
| favorite_count | Jumlah likes |
| full_text | Isi lengkap tweet |
| id_str | ID tweet |
| image_url | URL gambar pada tweet |
| in_reply_to_screen_name | Akun yang dibalas |
| lang | Bahasa tweet |
| location | Lokasi pengguna |
| quote_count | Jumlah quote tweet |
| reply_count | Jumlah balasan |
| retweet_count | Jumlah retweet |
| tweet_url | URL tweet |
| user_id_str | ID pengguna |
| username | Username pengguna |

---

## 📊 Findings
Hasil analisis menunjukkan bahwa dari **1.176 tweet** yang dianalisis:

- Sentimen positif mendominasi dengan **521 tweet (44,3%)**
- Sentimen netral sebanyak **437 tweet (37,2%)**
- Sentimen negatif sebanyak **218 tweet (18,5%)**

Distribusi ini menunjukkan bahwa mayoritas percakapan publik mengenai Old Trafford di platform X bersifat positif.

---

## ✅ Conclusion
Berdasarkan hasil analisis sentimen, persepsi publik terhadap Old Trafford di platform X cenderung positif. Mayoritas pengguna memberikan opini mengenai atmosfer pertandingan yang meriah, nilai sejarah stadion yang kuat, serta pengalaman menonton yang berkesan.

Hasil penelitian ini dapat menjadi bahan evaluasi bagi pihak terkait untuk mempertahankan kualitas stadion sekaligus meningkatkan pengalaman penggemar di masa mendatang.
